Welcome to the Babraham Bioinformatics helpdesk. You can use this system to put in a request for bioinformatics work, or to check on the progress of an existing support request.

To create a new support request, use the "New Job" link in the toolbar above. To check on the progress of an existing request enter the request ID code into the "Go to job" box, and press Go.

If you have any problems using this system, or have any suggestions as to how it could be improved then either enter it as a new support request, or send it to simon.andrews@babraham.ac.uk
